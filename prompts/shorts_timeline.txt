ภารกิจ: ออกแบบ JSON Blueprint สำหรับวิดีโอ 55 วินาที ที่มีจังหวะและอารมณ์เหมือนมนุษย์ตัดต่อ

=== PLATFORM: YouTube Shorts ===
- Total duration: 50–59 วินาที (ต้องไม่เกิน 60s)
- Jump cut ทุก ~4 วินาที (Shorts algorithm ต้องการ watch time สูง)
- Chapter structure: Hook / Value Content / CTA
- Hook 0–8s: สร้างคำถาม ดึงดูด ต้องการคำตอบ
- Value Content 8–45s: อธิบายสินค้า + social proof + ดราม่า
- CTA 45–55s: Subscribe + Link in description

=== OUTPUT RULES ===
1. ตอบด้วย JSON เท่านั้น — ห้ามมีข้อความอื่น ห้ามมี markdown code block
2. JSON ต้องผ่าน json.loads() ได้ทันที
3. ทุก t_start/t_end เป็นตัวเลข float (วินาที)
4. text_track ต้องมี style="pause" อย่างน้อย 2 entries
5. sfx_track ต้องมี trigger="jump_cut" ทุก jump cut
6. color_grade_map ต้องมีครบ 4 grades: vivid_pop, warm_bright, desaturated_dark, bw_sad
